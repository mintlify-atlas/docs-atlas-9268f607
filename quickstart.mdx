# Quickstart

Get started with RyunixJS and build your first application in minutes.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** 20, 22, or 24
- **npm**, **yarn**, or **pnpm**

## Create a New Project

<Steps>
  <Step title="Scaffold your application">
    Use the RyunixJS CLI to create a new project:

    <CodeGroup>
    ```bash npm
    npx @unsetsoft/cra@latest my-ryunix-app
    ```

    ```bash yarn
    yarn create @unsetsoft/cra my-ryunix-app
    ```

    ```bash pnpm
    pnpm create @unsetsoft/cra my-ryunix-app
    ```
    </CodeGroup>

    The CLI will prompt you to choose a template and configuration options.
  </Step>

  <Step title="Navigate to your project">
    ```bash
    cd my-ryunix-app
    ```
  </Step>

  <Step title="Start the development server">
    <CodeGroup>
    ```bash npm
    npm run dev
    ```

    ```bash yarn
    yarn dev
    ```

    ```bash pnpm
    pnpm dev
    ```
    </CodeGroup>

    Your application will start at `http://localhost:3000` with hot module replacement enabled.
  </Step>
</Steps>

<Note>
  The development server includes SSR by default, giving you instant feedback on both client and server rendering.
</Note>

## Your First Component

The CLI creates a basic application structure. Let's explore the main entry point:

```javascript app/index.ryx
import { createElement, render, useStore } from '@unsetsoft/ryunixjs'

function Counter() {
  const [count, setCount] = useStore(0)

  return createElement('div', { class: 'counter' },
    createElement('h1', null, 'Counter: ', count),
    createElement('button', {
      onClick: () => setCount(count + 1)
    }, 'Increment'),
    createElement('button', {
      onClick: () => setCount(count - 1)
    }, 'Decrement')
  )
}

render(createElement(Counter), document.getElementById('root'))
```

## Using JSX

RyunixJS supports JSX out of the box. Here's the same component with JSX syntax:

```jsx app/index.ryx
import { render, useStore } from '@unsetsoft/ryunixjs'

function Counter() {
  const [count, setCount] = useStore(0)

  return (
    <div className="counter">
      <h1>Counter: {count}</h1>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
      <button onClick={() => setCount(count - 1)}>
        Decrement
      </button>
    </div>
  )
}

render(<Counter />, document.getElementById('root'))
```

<Tip>
  RyunixJS automatically configures Babel to transform JSX using the RyunixJS runtime, so you don't need to import `createElement` when using JSX.
</Tip>

## Build Your Application

When you're ready to build for production:

<CodeGroup>
```bash npm
npm run build
```

```bash yarn
yarn build
```

```bash pnpm
pnpm build
```
</CodeGroup>

This creates optimized bundles in the `dist/` directory with:

- **Client bundle** - JavaScript for the browser
- **Server bundle** - Node.js bundle for SSR
- **Static HTML** - Pre-rendered pages (if using SSG)

## Available Scripts

Your RyunixJS project includes several useful commands:

- `npm run dev` - Start development server with HMR and SSR
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint on your code
